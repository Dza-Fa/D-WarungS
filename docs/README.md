## Quick Reference

### Technology Stack
- **Backend:** PHP 8.1+
- **Database:** MySQL 8.0+
- **Server:** Apache (XAMPP)
- **Framework:** Laravel 10.x (recommended)
- **Frontend:** Bootstrap 5.3, jQuery 3.7+

### User Roles
1. **Customer** - Browse, order, and track food
2. **Cashier** - Process payments, manage transactions
3. **Vendor** - Manage menu, fulfill orders
4. **Admin** - Full system management

### Development Phases
1. **Planning & Setup** (Week 1-2)
2. **Core Development - Backend** (Week 3-6)
3. **Core Development - Frontend** (Week 7-9)
4. **Advanced Features** (Week 10-12)
5. **Testing & Polish** (Week 13-14)
6. **Deployment** (Week 15-16)

### Key Features (MVP)
- User authentication & RBAC
- Vendor management
- Product/Menu management
- Shopping cart
- Order processing
- Payment handling
- Order tracking
- Vendor dashboard
